### Set up

To run this project, first download or clone this repo

create `.env.local` file and fill

```bash
    NEXT_PUBLIC_RAPID_API_KEY={YOUR rapidApi API KEY}
```

```bash
    // Install dependencies
    npm install
    // Run in development mode
    npm run dev
    // Build
    npm run build
    // Start Build site
    npm start
```
